## Common keybindings

|    **Key**     | **Functions As** |
| :------------: | :--------------: |
|      Esc       | freej2me Options |
|       Q        |   Options key    |
|       W        |     Back key     |
|       E        |        *         |
|       R        |        #         |
|   Number Row   |  Keypad Numbers  |
| Numpad's 7,8,9 |      1,2,3       |
| Numpad's 4,5,6 |      4,5,6       |
| Numpad's 1,2,3 |      7,8,9       |
